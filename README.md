#BookMedico
A Full-Stack MERN Health-Tech Platform

Overview
MediConnect is a centralized medical management system designed to bridge the gap between Patients, Doctors, and Pharmacy Admins (Store Owners). Unlike standard booking apps, this platform integrates the entire clinical workflow—from AI-assisted symptom awareness and appointment scheduling to digital prescription generation and automated inventory management.

Core Features
For Patients:

Automated Health Vault: Securely store digital prescriptions and manually upload hard copies of lab reports (PDF/Images).

Smart Scheduling: Real-time doctor availability check and instant appointment booking.

AI Integration: Advanced dashboard features for symptom analysis and health monitoring.

For Doctors:

Digital Prescription Pad: A dynamic interface to diagnose, prescribe medicines (with dosage/duration), and provide general advice.

Live Queue Management: Track daily appointments and update patient status (Confirmed, Cancelled, Completed) in real-time.

Patient History: Access a patient’s document vault to review previous medical history before consultation.

For Admins (Pharmacy/Store Owners):

Inventory Operations: A robust management system to track medicine stocks, handle categories (Tablets, Syrups, etc.), and monitor expiry dates.

Automated Alerts: Visual indicators for low-stock or out-of-stock critical medicines.

Financial Oversight: Dashboard analytics to track total revenue, active doctors, and patient footfall.

Technical Stack
Frontend: React.js, Tailwind CSS (Modern Glassmorphism UI), Lucide-React Icons.

Backend: Node.js, Express.js.

Database: MongoDB (Mongoose ODM).

Authentication: JWT (JSON Web Tokens) with role-based access control (RBAC).

Services: Custom Axios integration for seamless API communication.
